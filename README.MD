    ##PreRequisites : 
    Tech Stack : Express.Js, Node.Js, PostgresSQL, JWT


    ##Installation : 
    ###1 Install node modules
    <npm install>

    ###2 Library Install
    <npm install dotenv
    npm install --save-dev nodemon
    npm install cors>


    ###3 PostgresSQL Installation
    npm install pg 

    ###4 authentication
    npm install jsonwebtoken
    npm install bcrypt


    ###5 docs
    npm install swagger-ui-express swagger-jsdoc
